Markdown 入门语法

发表于 2016-01-04

1. 标题

文字前添加＃，共六级，每级对应几个＃，一级最大，六级最小
比如：
＃一级标题
＃＃ 二级标题
。。。
＃＃＃＃＃＃ 六级标题

示例：

一级标题

二级标题

三级标题

四级标题

五级标题
六级标题

2. 列表

无序列表：文字前加上－或者＊
有序列表：文字前加上1. 2. 3. 。。。

示例：

无序
  • 无序
  • 无序
  • 无序
有序
  1. 有序
  2. 有序
  3. 有序

3. 引用

文字前加上 > 这种尖括号

示例：

引用的部分文字

4. 链接及图片

插入链接和图片语法基本相同，差别在于！号
插入链接:[不知道](http://www.buzhidao.com)
插入图片:![不知道的图片](http://buzhidao.com/buzhidao_icon.png)

示例：

百度

5. 粗体与斜体

**粗体** 星号之间所包含的文字是粗体显示
*斜体* 星号之间所包含的文字是斜体

示例：

粗体

斜体
